## 编译说明 ##
**单文档编译** 

项目中相当多的代码都是只是单个源文件，或是单个源文件加一些头文件。可以
尝试使用下面的命令进行编译：

    `cc *.cc -o test`
    
**目录文档编译** 
部分测试代码是以目录的形式进行组织的，通常会提供有相应的`Makefile`编译
脚本。在安装有`make`的系统上应该可以使用如下命令：

    `make # 编译子项目`
    `make clean #清除项目中间文件。`
